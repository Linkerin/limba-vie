## Informal

| Romanian | English              |
| -------- | -------------------- |
| eu       | I                    |
| tu       | you                  |
| el       | he                   |
| ea       | she                  |
| noi      | we                   |
| voi      | you / y'all (plural) |
| ei       | they (masculine)     |
| ele      | they (feminine)      |

## Conjugation of the verb <span lang="ro-RO">_"a fi"_</span> (_"to be"_)

This verb is used to express existence, state, or identity.

| Pronume | a fi        |
| ------- | ----------- |
| eu      | sunt        |
| tu      | **e**ști    |
| el      | **e**ste    |
| ea      | **e**ste    |
| noi     | s**u**ntem  |
| voi     | s**u**nteți |
| ei      | sunt        |
| ele     | sunt        |

## Conjugation of the verb <span lang="ro-RO">_"a avea"_</span> (_"to have"_)

This verb is used to express possession or ownership.

| Pronume | a avea    |
| ------- | --------- |
| eu      | am        |
| tu      | ai        |
| el      | **a**re   |
| ea      | **a**re   |
| noi     | av**e**m  |
| voi     | av**e**ți |
| ei      | au        |
| ele     | au        |

<Attention heading="Note:">
  Personal pronouns (<span lang="ro-RO">_eu_</span>,{' '}
  <span lang="ro-RO">_tu_</span>, etc.) are often omitted in Romanian because
  the verb form clearly indicates the subject. <br />
  For example: instead of <span lang="ro-RO">_"Eu sunt obosit"_</span> (_"I am tired"_),
  you can simply say <span lang="ro-RO">_"Sunt obosit"_</span>.
</Attention>
