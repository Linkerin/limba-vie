# {metadata.title}

_Definite_ articles are attached to the noun as suffixes, whereas _indefinite_
articles stand as separate words.  
The form of the definite article also depends on whether the noun is
**singular** or **plural**.

## Masculin:

### Indefinite articles

| Singular                 | Plural                            |
| ------------------------ | --------------------------------- |
| **un** copil _(a child)_ | niște copi**i** _(some children)_ |
| **un** băiat _(a boy)_   | niște băieț**i** _(some boys)_    |
| **un** câine _(a dog)_   | niște câin**i** _(some dogs)_     |

### Definite articles

| Singular                  | Plural                      |
| ------------------------- | --------------------------- |
| copil**ul** _(the child)_ | copii**i** _(the children)_ |
| băiat**ul** _(the boy)_   | băieți**i** _(the boys)_    |
| câine**le** _(the dog)_   | câini**i** _(the dogs)_     |

## Feminin:

### Indefinite articles

| Singular               | Plural                         |
| ---------------------- | ------------------------------ |
| **o** fată _(a girl)_  | niște fet**e** _(some girls)_  |
| **o** carte _(a book)_ | niște cărț**i** _(some books)_ |
| **o** masă _(a table)_ | niște mes**e** _(some tables)_ |

### Definite articles

| Singular                | Plural                    |
| ----------------------- | ------------------------- |
| fat**a** _(the girl)_   | fete**le** _(the girls)_  |
| cart**ea** _(the book)_ | cărți**le** _(the books)_ |
| mas**a** _(the table)_  | mese**le** _(the tables)_ |

## Neutru:

### Indefinite articles

| Singular                   | Plural                              |
| -------------------------- | ----------------------------------- |
| **un** ou _(an egg)_       | niște ou**ă** _(some eggs)_         |
| **un** cântec _(a song)_   | niște cântec**e** _(some songs)_    |
| **un** telefon _(a phone)_ | niște telefoan**e** _(some phones)_ |
| **un** telefon _(a phone)_ | niște telefoan**e** _(some phones)_ |
| **un** grai _(a dialect)_  | niște grai**uri** _(some dialects)_ |

### Definite articles

| Singular                    | Plural                         |
| --------------------------- | ------------------------------ |
| ou**l** _(the egg)_         | ouă**le** _(the eggs)_         |
| cântec**ul** _(the song)_   | cântece**le** _(the songs)_    |
| telefon**ul** _(the phone)_ | telefoane**le** _(the phones)_ |
| telefon**ul** _(the phone)_ | telefoane**le** _(the phones)_ |
| grai**ul** _(the dialect)_  | graiuri**le** _(the dialects)_ |

<Attention heading="Note:">
  _Neuter_ nouns in Romanian act like **masculine** ones when **singular**, and
  like **feminine** ones when **plural**.
</Attention>
