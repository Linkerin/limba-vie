# {metadata.title}

The rules for deriving the plural form from the singular from a noun are quite
numerous. The plural from _has to be learned_ from the very beginning, as with
gender.

## Masculin

Most masculine nouns end in a _consonant_ or in _'-e'_. To form the plural, you
generally add _'-i'_ or _'-e'_ to the singular form:

| Singular                          | Plural                             |
| --------------------------------- | ---------------------------------- |
| un băiat _(a boy)_                | doi băieț**i** _(two boys)_        |
| un elev _(a pupil)_               | doi elev**i** _(two pupils)_       |
| un coleg _(a colleague)_          | doi coleg**i** _(two colleagues)_  |
|                                   |                                    |
| un frat<u>e</u> _(a brother)_     | doi fraț**i** _(two brothers)_     |
| un soar<u>e</u> _(a sun)_         | doi sor**i** _(two suns)_          |
| un peșt<u>e</u> _(a fish)_        | doi peșt**i** _(two fish)_         |
|                                   |                                    |
| un le<u>u</u> _(a lion)_          | doi le**i** _(two lions)_          |
| un fi<u>u</u> _(a son)_           | doi fi**i** _(two sons)_           |
| un ministr<u>u</u> _(a minister)_ | doi miniștr**i** _(two ministers)_ |

Note:

| Singular             | Plural                         |
| -------------------- | ------------------------------ |
| un om _(a human)_    | doi oamen**i** _(two humans)_  |
| un copil _(a child)_ | doi copi**i** _(two children)_ |
| un cal _(a horse)_   | doi ca**i** _(two horses)_     |
| un tată _(a dad)_    | doi taţ**i** _(two dads)_      |

## Feminine

Most feminine nouns end in _'-ă'_ or _'-e'_. To form the plural, you generally
replace the singular ending with _'-e'_ or _'-i'_:

| Singular                      | Plural                            |
| ----------------------------- | --------------------------------- |
| o cas<u>ă</u> _(a house)_     | două cas**e** _(two houses)_      |
| o fat<u>ă</u> _(a girl)_      | două fet**e** _(two girls)_       |
| o mas<u>ă</u> _(a table)_     | două mes**e** _(two tables)_      |
|                               |                                   |
| o sear<u>ă</u> _(an evening)_ | două ser**i** _(two evenings)_    |
| o școal<u>ă</u> _(a school)_  | două școl**i** _(two schools)_    |
| o sal<u>ă</u> _(a hall)_      | două săl**i** _(two halls)_       |
|                               |                                   |
| o cefe<u>a</u> _(a coffee)_   | două cafe**le** _(two coffees)_   |
| o pijam<u>a</u> _(a pyjama)_  | două pijama**le** _(two pyjamas)_ |
| o feme<u>ie</u> _(a woman)_   | două feme**i** _(two ministers)_  |

Note:

| Singular       | Plural                    |
| -------------- | ------------------------- |
| o zi _(a day)_ | doi zi**le** _(two days)_ |

## Neutru

Neuter nouns often end in a _consonant_, _'-u'_, _'-iu'_, or _'-o'_ in the
singular form. To form the plural, you generally replace the singular ending
with '-uri':

| Singular                          | Plural                             |
| --------------------------------- | ---------------------------------- |
| un oraș _(a town)_                | două oraș**e** _(two towns)_       |
| un salon _(a lounge)_             | două saloan**e** _(two lounges)_   |
| un creion _(a pencil)_            | două creioan**e** _(two pencils)_  |
|                                   |                                    |
| un joc _(a game)_                 | două joc**uri** _(two games)_      |
| un câmp _(a field)_               | două câmp**uri** _(two fields)_    |
|                                   |                                    |
| un metro<u>u</u> _(a metro)_      | două metro**uri** _(two metros)_   |
| un lucr<u>u</u> _(a thing)_       | două lucr**uri** _(two things)_    |
|                                   |                                    |
| un teatr<u>u</u> _(a theater)_    | două teatr**e** _(two theaters)_   |
| un muze<u>u</u> _(a museum)_      | două muze**e** _(two museums)_     |
|                                   |                                    |
| un fotoli<u>u</u> _(an armchair)_ | două fotoli**i** _(two armchairs)_ |
| un studi<u>u</u> _(a study)_      | două studi**i** _(two studies)_    |
|                                   |                                    |
| un radi<u>o</u> _(a radio)_       | două radi**ouri** _(two radios)_   |
| un domin<u>o</u> _(a domino)_     | două domin**ouri** _(two dominos)_ |

Note:

| Singular        | Plural                    |
| --------------- | ------------------------- |
| un ou _(a egg)_ | două ou**ă** _(two eggs)_ |
